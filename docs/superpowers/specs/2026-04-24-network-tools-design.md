# 网络工具套件设计文档

**日期**: 2026-04-24
**范围**: 7 个网络工具（子网计算器、进制转换器、MAC 地址查询、IPv6 格式转换、ASN 查询、DNS 解析、Whois 查询）

---

## 1. 子网计算器

### 功能
- 输入 IP 地址 + 前缀长度，计算网络信息

### 输出信息
- 网络地址、广播地址、子网掩码
- 可用 IP 范围、主机数量
- CIDR 表示法、子网数量

### VLSM 可视化（三种模式可切换）
- **树状层级图**: 显示子网的父子关系，层层嵌套
- **表格 + 色块**: 用表格展示每个子网，用颜色区分可用/已用
- **IP 段拆分器**: 输入一个大段，手动切分成多个小子网，每个段用色块表示

### 实现
- 纯前端计算

---

## 2. 进制转换器

### 功能
- IP 模式: 十进制 ↔ 十六进制 ↔ 二进制互转
- 普通进制: 十六进制/十进制/八进制/二进制互转

### 特点
- 实时转换
- 高亮对应位

### 实现
- 纯前端计算

---

## 3. MAC 地址查询

### 功能
- 输入 MAC 地址，查询厂商信息

### 输出
- 厂商名称、地址、国家

### API
- free-api.com (maclookup.app/api/v2/mac)

### 实现
- 客户端请求外部 API

---

## 4. IPv6 格式转换

### 功能
- 压缩/扩展互转
- EUI-64 转换

### 输入
- 任意格式 IPv6 地址

### 输出
- 所有格式的转换结果

### 实现
- 纯前端计算

---

## 5. ASN 查询

### 功能
- 输入 ASN 号，查询所属组织

### 输出
- 组织名称
- IP CIDR 块列表

### API
- iphub.info 或类似免费 API

### 实现
- 客户端请求外部 API

---

## 6. DNS 解析

### 功能
- 查询域名的 DNS 记录

### 支持记录类型
- A、AAAA、MX、TXT、CNAME、NS、SOA、SRV、PTR

### 输出
- 表格形式显示各记录结果

### API
- dns.google (DNS over HTTPS)

### 实现
- 客户端请求外部 DoH API

---

## 7. Whois 查询

### 功能
- 查询域名的注册信息

### 输出
- 注册商
- 注册日期、到期日期
- 注册人/公司
- DNS 服务器
- 域名状态

### API
- whoisjsonapi.com 或 whois.marketing

### 实现
- 客户端请求外部 API

---

## 统一交互设计

### 导航
- 左侧工具栏（与现有工具统一）

### 页面结构
- 每个工具独立页面

### 通用功能
