# 正则表达式测试工具

一个简单、美观的在线正则表达式测试工具，支持实时匹配、高亮显示和常用正则示例。

## 功能特性

- 🎨 实时正则表达式匹配和高亮
- 📦 预置常用正则表达式示例
- 🔍 详细的匹配结果展示
- 🎯 支持所有 JavaScript 正则修饰符
- 📱 响应式设计

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署到 Vercel

### 方法一：使用 Vercel CLI

```bash
# 安装 Vercel CLI
npm i -g vercel

# 登录并部署
vercel
```

### 方法二：Git 仓库部署

1. 将代码推送到 GitHub/GitLab 仓库
2. 在 [vercel.com](https://vercel.com) 导入仓库
3. 点击部署即可

## 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Vercel (部署)
